# 🎓 Student Management System

A simple Student Management System built with Python using Object-Oriented Programming (OOP).

This project was created to practice Python fundamentals such as classes, objects, methods, loops, lists, and user interaction through a command-line interface.

---

## ✨ Features

- ➕ Add new students
- 📋 Display all students
- 🔍 Search students by name
- ❌ Mark a student as absent
- ✅ Mark a student as present
- 👨‍🎓 Store student information:
  - Name
  - Age
  - Major
  - Attendance Status

---

## 🛠 Technologies

- Python 3
- Object-Oriented Programming (OOP)

---

## 📂 Project Structure

